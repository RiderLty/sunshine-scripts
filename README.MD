# 快速设置显示器模式

多显示器模式 

1=仅内置屏幕

2=仅外置屏幕

3=扩展

4=复制


## 直接调用
### 使用内置模式
向 typeMap 内添加模式

格式为

```
名称:[多显示器模式,宽度,高度,刷新率]
```

然后可以直接通过单个名称参数来调用设置

## 手动设定

使用 多显示器模式 宽度 高度 刷新率 四个参数 直接设定

## 服务器模式
用于远程调用

* 使用内置模式
```
/set/{type}
```
type为内置参数的名称

例如：/set/default


* 手动设定
```
/dwhr/{mode}
```
多显示器模式,宽度,高度,刷新率  用.连接

例如：/dwhr/1.1920.1080.120
